# Synthetix Intelligence

Synthetix Intelligence is a sophisticated content architecture engine designed to streamline the creation of high impact social media strategies. Leveraging advanced artificial intelligence, this platform transforms abstract concepts into polished, platform specific narratives helping creators and brands establish a commanding digital presence.

Designed with a focus on minimalism and efficiency, the interface provides a distraction free environment where strategy meets execution. It allows users to generate tailored drafts for multiple platforms simultaneously, ensuring consistency in voice while adapting to the unique nuances of each network.

## Core Capabilities

The engine is built to handle the complete lifecycle of a social media post, from ideation to visual direction.

- **Multi Platform Generation**
  Create optimized content for LinkedIn, Twitter, Instagram, and more from a single prompt. The system automatically adjusts length, formatting, and style to fit the medium.

- **Tone Calibration**
  Refine the voice of your content with precision. Whether you need a professional, witty, or urgent tone, the engine adapts its output to match your strategic intent.

- **Visual Synthesis**
  Beyond text, the system generates accompanying visual assets. It suggests engaging imagery and produces high quality renders in various aspect ratios to complement your written content.

## Getting Started

Follow these instructions to set up the project on your local machine for development and testing purposes.

### Prerequisites

Ensure you have Node.js installed on your system. This project uses standard web technologies and requires a modern runtime environment.

### Installation

1.  Clone the repository to your local machine.
2.  Navigate to the project directory in your terminal.
3.  Install the necessary dependencies by running the install command:

    ```bash
    npm install
    ```

### Running the Application

To start the local development server, execute the following command:

    ```bash
    npm run dev
    ```

The application will initialize and be accessible at the local address provided in your terminal, typically strictly on localhost port 5173.

## Building for Production

When you are ready to deploy, you can generate an optimized build of the application. This process compiles the code and assets for production environments.

    ```bash
    npm run build
    ```

## Technology Stack

This project is architected using a modern, performance oriented stack:

- **React** for the user interface
- **TypeScript** for type safety and code reliability
- **Vite** for rapid development and bundling
- **Tailwind CSS** for styling and design system implementation
